## Installation

### Create a virtual environment 

`python3 -m venv env`

### Activate the virtual environment
 
For macOS/Linux

`source env/bin/activate`

For windows

`env\Scripts\activate`

### Install dependencies

`pip install -r requirements.txt`


### Run the app 

`python manage.py runserver`
